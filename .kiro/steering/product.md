# Product Overview

## Jinbao Protocol - RWA DeFi 4.0

Jinbao Protocol is a Real World Asset (RWA) based decentralized finance protocol featuring an automated token burning mechanism. It implements a dual-token deflationary model with MC (Master Coin) and JBC (Jinbao Coin) tokens.

### Core Features

- **Ticket System**: 4-tier ticket levels (100/300/500/1000 MC) with liquidity requirements
- **Liquidity Mining**: 3 cycle options (7/15/30 days) with daily yields of 2.0%-3.0%
- **Multi-tier Reward System**: 20-level referral chain with V0-V9 team levels
- **Automated Burning**: Daily token burning via Cloudflare Pages + GitHub Actions
- **AMM Trading**: Built-in automated market maker for MC/JBC token swaps

### Key Components

- **Smart Contracts**: Complete DeFi protocol implementation with upgradeable architecture
- **Frontend Application**: Modern React + TypeScript + Vite web application
- **Automated Burning**: Cloudflare Functions for scheduled token burning
- **Real-time Monitoring**: Complete transaction history and data analytics

### Target Networks

- **MC Chain** (Primary): Chain ID 88813
- **Sepolia Testnet**: For testing
- **BSC Testnet**: Alternative testnet

The protocol focuses on sustainable tokenomics through controlled inflation, liquidity incentives, and automated deflationary mechanisms.
